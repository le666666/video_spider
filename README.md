# Video_spider

## 请求示例

支持GET/POST `url`参数必填

## 返回数据

| 字段名 | 说明 | 字段名 | 说明 |字段名 | 说明 |字段名 | 说明 |
|  ----  | ----  | ----  | ---- |---- |---- |----|----|
| author | 视频作者| avatar | 作者头像 | like | 视频点赞量 | time | 视频发布时间 |
| title | 视频标题 | cover | 视频封面 | url | 视频无水印链接 | sex  | 作者性别 |
| age | 作者年龄 | city | 所在城市 | uid | 作者id | code | 状态码 |
